Envoy Thrift Java
========


Publish
------------
This project needs to be published to artifactory.

``` bash
$ mvn deploy
```

Import
------------
Maven
``` xml
<dependency>
    <groupId>com.sazze.en</groupId>
    <artifactId>envoy-thrift-java</artifactId>
    <version>VERSION</version>
</dependency>
```

sbt
```
"com.sazze.en" % "envoy-thrift-java" % "VERSION"
```
